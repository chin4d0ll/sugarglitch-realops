## 🚀 การติดตั้ง

### 1. Clone Repository
```bash
git clone <repository-url>
cd sugarglitch-realops
```

### 2. ติดตั้ง Dependencies
```bash
pip install -r requirements.txt
```

### 3. Setup โปรเจค
```bash
python project_setup.py
```

```bash
cp .env.template .env
# แก้ไขไฟล์ .env ตามต้องการ
```
